# Git Finder 🔍

A sleek web application to search and view GitHub profiles by username.

## 🚀 Features

- 🔍 Search for any GitHub user
- 📊 View profile information: bio, location, followers, etc.
- 📁 List user's repositories
- 🌐 Fully responsive UI

## 🛠️ Tech Stack

- Javascript
- GitHub REST API
- CSS
- HTML
